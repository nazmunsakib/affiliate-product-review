== Installation ==

= Minimum Requirements =

* PHP 7.3 or greater is recommended
* MySQL 5.6 or greater is recommended

You can install the affiliate product review from your WordPress Dashboard or manually upload it through FTP.

= OPTION 1: Install the Affiliate Product Review Plugin from WordPress Dashboard =

1. Navigate to Plugins -> Add New.
2. Search for 'Affiliate Product Review' and click on the Install button to install the plugin.
3. Activate the plugin in the Plugins menu.
4. Add add a new post look at the post editor bottom Check the box 'Is Product Review ?' then click the button 'Add New Product Review'

= OPTION 2: Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: affiliate-Product-eview.zip.
2. Upload the 'affiliate-Product-eview.zip' file to your '/wp-content/plugins' directory.
2. Unzip the file affiliate-Product-eview.zip (do not rename the folder).
